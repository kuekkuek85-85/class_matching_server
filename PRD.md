# PRD.md (해커톤용 요약 버전)

## 🎯 프로젝트 목표
구글 폼 없이 웹에서 바로 프로그램 신청/배치를 처리하는 1일 해커톤 MVP 시스템.

- 학생: `/` 에서 학번/이름 → 지망 선택 → 제출
- 교사: `/admin` 에서 프로그램 등록, 신청 현황, 배치 실행, CSV 다운로드
- 중복 제출 시: 전화번호 + 생년월일 추가 입력 후 재제출

---

## 🧑‍🎓 학생 시나리오
1. `/` 진입
2. 학번 + 이름 입력
3. 1지망~3지망 선택
4. 제출
5. 이미 제출한 학번이면 → 전화번호 + 생년월일 필수 입력 → 재제출 가능

---

## 👩‍🏫 교사 시나리오
1. `/admin` 접속
2. 프로그램 등록 `/admin/programs`
3. 신청 현황 확인 `/admin/allocation`
4. 기본 배치 실행
5. 결과 통계 확인 + CSV 다운로드

---

## 📌 주요 기능 (MVP)
- 프로그램 관리 (등록/삭제)
- 학생 신청/재신청
- 중복 제출 보호 로직
- 기본 배치 알고리즘
- CSV 다운로드

---

## ⛔ 제외 범위
- 구글 폼/스프레드시트 연동
- 인증/로그인 시스템
- 고급 AI 배치 (선택사항)
